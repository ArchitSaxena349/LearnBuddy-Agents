This folder contains the backend for LearnBuddy, implemented in Python.

Prerequisites
- Python 3.9 or newer
- Git (optional)

Quick start (PowerShell)

1. Create and activate a virtual environment:

   python -m venv .venv
   .\.venv\Scripts\Activate.ps1

2. Install dependencies:

   pip install -r requirements.txt

3. Run the backend:

   python main.py

Notes
- If your environment uses Bash/macOS/Linux, activate the venv with:
  source .venv/bin/activate
- If the project uses environment variables, create a `.env` file or export them before running.

Stopping
- Use Ctrl+C in the terminal to stop the running process.

More
- See `requirements.txt` for Python dependencies and `main.py` for the entrypoint and configuration.
